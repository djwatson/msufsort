MSufSort suffix sorting algorithm.

Authors: Michael A. Maniscalco and Simon J. Puglisi
Date: May 10, 2007

web: http://www.michael-maniscalco.com

-------------------------------------------------------
Usage:

// forward BWT on file myFile.txt to file myFile.bwt
MSufSort.3.1.1.exe b myFile.txt myFile.bwt

// reverse BWT on file myFile.bwt to file myFileBackup.txt
MSufSort.3.1.1.exe u myFile.bwt myFileBackup.txt

// suffix array on file myFile.txt to file myFile.sa
MSufSort.3.1.1.exe s myFile.txt myFile.sa

// help
MSufSort.3.1.1.exe ?|h
